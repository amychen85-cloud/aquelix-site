# AQUELIX Website

Premium B2B faucet and shower website for Cloudflare Pages / Codex.

## Project type
Static website: HTML + CSS + JS.

## Pages
- `index.html` — homepage
- `products.html` — product catalog / RFQ entry
- `technology.html` — quality and technology page
- `about.html` — brand story
- `contact.html` — RFQ contact form

## Deployment
Use Cloudflare Pages:
1. Connect this GitHub repository.
2. Framework preset: None / Static HTML.
3. Build command: leave empty.
4. Output directory: `/` or leave empty.
5. Deploy.
